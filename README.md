Technologies used:
* Nextjs
* tailwindcss
* json server
* zustand
    * simplicity
    * typescript support
    * no context providers needed
    * performance

## Getting Started

How to run:

```bash
npm run dev:full
```

Open [http://localhost:3001](http://localhost:3001) with your browser to see the result.
